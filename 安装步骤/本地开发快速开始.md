# 本地开发快速开始

在本机运行编辑端 + API，用于日常开发与调试。

## 环境要求

- Node.js **20.19+** 或 **22.12+**（见项目 `package.json` / `scripts/check-node.mjs`）

## 步骤

### 1. 进入项目目录

```bash
cd /path/to/Cat2
```

### 2. 配置环境变量

```bash
cp .env.example .env
```

编辑 `.env`，至少修改：

- `JWT_SECRET` — 随机长字符串
- `ADMIN_PASSWORD` — 管理员登录密码

### 3. 安装依赖

```bash
npm install
```

### 4. 启动开发服务（推荐）

**Windows：**

```bash
npm run dev:local
```

会自动检查 Node 版本、结束占用 3001 的旧 API，并同时启动 API + Vite。

**其它系统：**

```bash
npm run dev
```

### 5. 打开页面

| 页面 | 地址 |
|------|------|
| 登录 | http://localhost:5173/login.html |
| 编辑 | http://localhost:5173/ |
| 公众页 | http://localhost:5173/viewer.html |
| 表格模板库 | http://localhost:5173/table-templates.html（或后台 **设置 → 表格模板库**） |
| API | http://localhost:3001 |

默认用户名 `admin`，密码为 `.env` 中的 `ADMIN_PASSWORD`。

## 常用命令

| 命令 | 说明 |
|------|------|
| `npm run dev:local` | 开发（含清理旧 API，Windows 推荐） |
| `npm run dev:web` | 仅前端 |
| `npm run dev:server` | 仅 API |
| `npm run build` | 构建 `dist/` |
| `npm start` | 仅启动 API（需先 build 或配合 dist） |
| `npm run reset:admin` | 将管理员密码重置为 `.env` 中的值 |

## 数据位置

| 路径 | 内容 |
|------|------|
| `data/cat.db` | 证书、SVG/表格模板、用户等（SQLite） |
| `data/uploads/` | 表格中的上传图片 |
| `layout-settings.json` | 编辑框布局本地备份 |
